Smart Home System - Scenario

SECTION B - SCENARIO: Smart Home System

A smart home system is controlled automatically based on certain measures,such as water leakage, short circuit , Gas Leakage and fire at home .These are monitored and measured using various sensors .

Based on the certain combinations of these measures , the automatic control system will send a emergency message to the owner based on the following measures.

  • Water leakage (W=1, if any leakage; otherwise 0)
  • Short circuit (C=1 if any shortage; otherwise 0)
  • Gas leak (G=1, if any leakage; else 0)
  • Fire buzzer (F=1; if fire is detected (on); 0, if fire is not detected(Off))

The following the conditions for sending emergency message are:

  1. Gas leak and fire buzzer is on
  2. Only the fire buzzer is on
  3. Water leakage at any place
  4. short circuit and the fire buzzer is on

The automatic control system will send a normal message to the owner based on the following measures:

  • Bell (B=1, if its rings; otherwise 0)
  • Motion sensor (M=1 if motion is detected; else 0)
  • Post box (P=1, if post is received; else 0)
  • Door (D=1, if open; else 0, if closed)

The following the conditions for sending normal messages are:

  1. Motion is detected and the door is closed
  2. Door is closed, bell is ringed, and post is received
  3. Bell does not ring, door is open, and motion is detected

The system will provide two outputs such as Emergency (A) and Normal(B). Based on these outputs a message will be sent to the fire station.

The condition for sending a message to the fire station is:

  • If A=1 and B=0
  • If A=1 and B=1
